## 出现的问题

对 `YAML` 不太了解，总是不出效果

最后在[这里](http://www.cnblogs.com/kaiye/archive/2013/04/24/3039346.html)找到了答案：

>   -   `|` 分隔符前面有一个空格，按照规范所有 `:` 符号声明后必须有一个空格；
>   -   新起一行的代码块，至少必须有一个缩进符，且缩进符为`空格`，不能为 tab 制表符等其他空白。
